# 🎬 Data Analysis & Visualization Project

A simple and powerful **Python CLI-based Data Analyzer** that allows you to load, explore, clean, analyze, and visualize CSV datasets easily 📊

---

## ✨ Features

✅ Load CSV dataset
✅ Explore data (head, tail, columns, info)
✅ Perform DataFrame operations
✅ Handle missing values
✅ Generate statistics
✅ Create multiple visualizations (Bar, Line, Scatter, Pie, Histogram, Stack)
✅ Save visualizations as image files

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas 📊
- Matplotlib 📉

## 🚀 How to Run the Project

1. Install required libraries:
```

pip install pandas matplotlib

```

2. Run the program:
```

python main.py

```

3. Follow menu options step by step:
- Load dataset
- Explore data
- Perform operations
- Visualize data
